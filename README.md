# Virtual Mouse

## Installation

## Clone the Repository
```bash
git clone https://github.com/whitehatboy005/Virtual-Mouse
```
## Move the file
```bash
cd Virtual-Mouse
```
## Install Dependencies
```bash
pip install -r requirements.txt
```
## Run the Program
```bash
python mouse.py
```
